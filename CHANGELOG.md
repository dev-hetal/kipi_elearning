# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-25

### Added
- Initial release of Kipi E-Learning package
- Course management screens (Explore, My Courses, Course Details)
- Course enrollment with wallet integration
- Course index/chapter content viewer
- Teacher index merging functionality
- Reviews and ratings system
- Multi-language support (English, Gujarati)
- Provider interfaces for dependency injection
- GetX state management and routing
- Reusable widgets (CourseCard, IndexTreeView, etc.)
- Theme customization support
- Localization with fallback chain
- Course manager singleton for state management
- Utility functions for course grouping
